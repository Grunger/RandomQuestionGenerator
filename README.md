## Описание программы
**Random Quote Generato** – это простое desktop-приложение для создания цитат

## Возможности программы
- Добавление цитат с указанием автора, темы и текста
- Просмотр всех записей в виде таблицы
- Фильтрация записей:
  - по автору
  - по теме
- Удаление записей из списка
- Автоматическое сохранение данных в JSON-файл (`quotes.json`)
- Загрузка ранее сохранённых записей при запуске
- Валидация вводимых данных с понятными сообщениями об ошибках

## Проверка корректности ввода
- **Автор** – не должно быть пустым. Минимальная длина не проверяется, но поле обязательно для заполнения.
- **Тема** – не должно быть пустым. Минимальная длина не проверяется, но поле обязательно для заполнения.
- **Текст** – не должно быть пустым. Минимальная длина не проверяется, но поле обязательно для заполнения.

4. Откройте файл:
